# Career Console

A personal project focused on building structured, developer-first tools powered by React and TypeScript.

## Resume Builder

A custom resume builder designed to generate clean, ATS-friendly resumes with real-time editing and print-optimized PDF output.

## Features

- Dynamic resume editing with localStorage persistence
- Structured data model for experience, skills, and projects
- Real-time preview rendering
- Print-optimized layout for consistent PDF export

## Tech Stack

- React
- TypeScript
- Tailwind CSS

This project was built to explore structured data systems and practical UI tools while reinforcing component architecture, state management, and real-world frontend workflows.
